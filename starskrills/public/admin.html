Pasos para desplegar en Railway
1. Crea cuenta gratis en railway.app
2. Crea cuenta en github.com
3. Sube la carpeta starskrills a un repo de GitHub
4. En Railway → New Project → Deploy from GitHub
5. Variables → agrega ADMIN_SECRET=tu_clave
6. Settings → Generate Domain → copia la URL
7. Pega la URL aquí arriba y ¡listo! 🚀
